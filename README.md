# Bramp Landing Page

A responsive landing page for Bramp, a platform for buying, selling, and trading cryptocurrency and gift cards.

## Features

- Responsive design that works on all device sizes
- Modern and clean UI
- Information about the platform's key features and capabilities
- FAQ section
- Call-to-action buttons for signing up and downloading mobile apps

## Project Structure

- `landing-page.html` - Main HTML file
- `css/` - Contains all styling files
  - `landing-page.css` - Main stylesheet
  - `globals.css` - Global variables and styles
  - `styleguide.css` - Design system styles
- `img/` - Contains all images used in the project

## Getting Started

To run this project locally:

1. Clone the repository
2. Open the project folder
3. Start a local server (e.g., `python -m http.server 8000`)
4. Open your browser and navigate to `http://localhost:8000`
